## 二、详细指标清单

### ✅ 理杏仁可以获取的指标（10个）

#### 1. 盈利能力指标（2个）

**1.1 行业平均ROE** ✓
- **获取路径**：理杏仁 → 指数估值 → 申万行业 → ROE
- **数据特点**：
  - 提供TTM数据
  - 可查看历史走势图
  - 可查看历史分位数
  - 更新频率：季度更新
- **VIP功能**：历史数据导出

**1.2 行业平均毛利率** ✓（部分）
- **获取路径**：理杏仁 → 行业数据 → 申万行业 → 盈利能力 → 毛利率
- **注意**：
  - 免费版：仅显示最新数据
  - VIP版：可查看历史数据和趋势
  - 数据完整性不如Wind

---

#### 2. 成长性指标（2个）

**2.1 行业收入增速** ✓
- **获取路径**：理杏仁 → 行业数据 → 申万行业 → 营收同比
- **数据特点**：
  - TTM同比增速
  - 可查看历史趋势
  - 更新频率：季报后更新
- **替代方案**：也可通过"指数估值"页面的相关指标推算

**2.2 行业利润增速** ✓
- **获取路径**：理杏仁 → 行业数据 → 申万行业 → 净利润同比
- **数据特点**：
  - TTM同比增速
  - 可用于计算PEG
  - 季度更新

---

#### 3. 现金流指标（1个）

**3.1 经营性现金流/净利润** ✓（VIP）
- **获取路径**：理杏仁VIP → 行业财务分析 → 现金流指标
- **注意**：
  - 免费版无此指标
  - VIP版提供行业整体数据
  - 可查看个股再汇总

---

#### 4. 估值指标（2个）

**4.1 行业PE/PB历史分位** ✓✓（核心优势）
- **获取路径**：理杏仁 → 指数估值 → 申万行业指数 → PE/PB历史分位
- **数据特点**：
  - **免费**且功能强大
  - PE-TTM、PE-LYR、PB历史分位
  - 可选1年/3年/5年/10年/全部
  - 可视化图表优秀
  - 实时更新（每日）
- **这是理杏仁的王牌功能！**

**4.2 PEG（市盈率相对盈利增速比）** ✓
- **获取路径**：
  - PE：理杏仁 → 申万行业 → PE-TTM
  - 增速：理杏仁 → 申万行业 → 净利润同比
  - 需手工计算：PEG = PE / 增速
- **注意**：理杏仁不直接提供PEG，需自己算

---

#### 5. 红线触发指标（2个）

**5.1 估值历史高位** ✓✓
- **获取路径**：理杏仁 → 申万行业 → PE/PB分位（5年）
- **监控方法**：
  - 直接查看当前分位数
  - 设定 > 95%分位为红线
  - 免费可用，实时监控
- **这是理杏仁最适合的场景**

**5.2 毛利率连续下滑** ✓（VIP）
- **获取路径**：理杏仁VIP → 行业数据 → 毛利率历史数据
- **监控方法**：
  - 导出近8个季度数据
  - 人工判断是否连续下滑
- **免费版无历史数据**

---

### ❌ 理杏仁无法获取的指标（16个）

#### 1. 竞争格局指标（3个）

**❌ 1.1 行业集中度（CR5）**
- **理杏仁无此数据**
- **替代方案**：
  - 手工计算：东方财富查行业公司 → 按营收排序 → 自己算
  - 或查券商研报

**❌ 1.2 龙头市场份额变化**
- **理杏仁无此数据**
- **替代方案**：
  - 查上市公司年报"市场地位"章节
  - 或同花顺免费版F10

**❌ 1.3 价格体系稳定性**
- **理杏仁无此数据**
- **替代方案**：
  - 卓创资讯、生意社（部分免费）
  - 国家统计局CPI/PPI分项

**✓ 1.4 产能利用率**
- **理杏仁有个股产能利用率（VIP）**
- **行业整体数据**：需查统计局或行业协会

## 四、理杏仁使用技巧

### 4.1 快速查询行业估值分位

```
步骤：
1. 访问 lixinger.com
2. 点击"指数估值"
3. 选择"申万行业"
4. 找到目标行业（如"食品饮料"）
5. 查看"PE分位"和"PB分位"（建议看5年分位）

快捷判断：
- 分位 < 30% → 低估 ✓
- 分位 30-50% → 合理
- 分位 > 70% → 高估 ✗
```

---

### 4.2 查询行业ROE趋势

```
步骤：
1. 指数估值 → 申万行业 → 选择行业
2. 下拉找到"ROE-TTM"指标
3. 点击图表查看历史走势

分析要点：
- ROE是否处于上升趋势？
- 当前ROE在历史什么位置？
- 与全A中位数比较如何？
```

---

### 4.3 批量下载个股数据（VIP）

```
步骤：
1. 理杏仁VIP → 数据浏览器
2. 筛选条件：申万行业 = "XX行业"
3. 选择字段：营收、净利润、ROE、存货等
4. 导出Excel
5. 自己计算行业整体指标（加权平均）

适用场景：
- 计算CR5（前5名营收占比）
- 计算行业存货同比
- 计算资本支出强度
```

---

### 4.4 设置估值预警（手动方案）

理杏仁暂无自动预警功能，建议：
```
1. 每月1号手动查看关注行业的PE/PB分位
2. 记录在Excel表格中
3. 设置条件格式：
   - 分位 > 80% 标橙色（预警）
   - 分位 > 95% 标红色（红线）
4. 或者用理杏仁API（VIP）自己写脚本监控
```

## 六、总结与建议

### 6.1 理杏仁适用场景

**✓ 最适合**：
1. 行业估值分位监控（核心功能）
2. 行业盈利能力分析（ROE、毛利率）
3. 行业成长性评估（收入/利润增速）
4. 个股财务数据批量导出（VIP）

**✗ 不适合**：
1. 宏观景气度分析（无PMI、PPI等）
2. 竞争格局分析（无CR5、市场份额）
3. 资金流向监控（无北向资金数据）
4. 高频价格监控（无商品价格数据）

---

### 6.2 免费渠道最优组合

**方案一：纯免费**
- 理杏仁免费版（估值、ROE）
- 统计局/央行（PMI、PPI、社融）
- 东方财富（资金流向）
- 生意社（价格，部分）
- **覆盖率：70%**

**方案二：最优性价比（推荐）**
- 理杏仁VIP（365元/年）
- 统计局/央行（免费）
- 东方财富（免费）
- **覆盖率：85%**

**方案三：专业版**
- Choice经济版（5000元/年）
- 理杏仁VIP（365元/年）
- **覆盖率：95%**
