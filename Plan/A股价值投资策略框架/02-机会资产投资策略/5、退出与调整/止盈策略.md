# 止盈策略（核心策略）

## 目标价止盈（主要退出方式）
```
涨幅>50% 且 PE回到历史中位数 → 分批卖出
涨幅>80% → 直接卖出，不贪心
```

## 分批卖出计划
```
涨幅达50%: 卖出30%（锁定部分利润）
涨幅达70%: 卖出40%（保留少量仓位）
涨幅达90%: 全部清仓（获利了结）
```

## 时间止盈
```
持有满18个月，涨幅>30% → 分批卖出
持有满24个月，无论盈亏 → 重新评估是否继续持有
```

## 催化剂兑现止盈
```
买入时的催化剂已完全兑现 → 卖出
估值修复到位（PE回到中位数） → 卖出
```

## 机会成本止盈
```
发现更好的错杀标的（预期收益差>30%） → 换仓
```
