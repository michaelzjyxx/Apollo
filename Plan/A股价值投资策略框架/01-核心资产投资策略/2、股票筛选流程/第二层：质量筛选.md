# 第二层：质量筛选 → 核心资产池（30-50只候选）

## 财务指标（硬性，全部满足）
```yaml
ROE: ≥15%（连续5年）
毛利率: ≥40%（连续3年）
营收增长: 年均≥10%
负债率: <60%
现金流: 经营性现金流/净利润 >0.8
```

## 护城河（至少2项）
- 品牌护城河（消费品）
- 技术护城河（专利、研发占比>8%）
- 网络效应（平台、渠道）
- 成本优势（规模经济）
- 转换成本（客户粘性高）

## 行业地位
- 细分行业前3名
- 市占率≥15%或持续提升

## 管理层
- 创始人/核心团队持股>5%
- 无重大违规记录
- 分红率>30%或持续回购
