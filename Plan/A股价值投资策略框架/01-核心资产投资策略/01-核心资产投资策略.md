# 核心资产投资策略

**定位**：占总仓位70%，长期持有优质公司
**目标**：通过5-10年持有期，获得年化12%的复利增长
**核心理念**：以合理价格买入伟大公司，享受企业成长红利

---

## 一、核心资产定义

### 资产特征
- **优质公司**：ROE≥15%，护城河深厚，行业地位稳固
- **长期视角**：持有周期5-10年，不因短期波动卖出
- **合理估值**：不过分强调便宜，PE处历史中位数即可买入
- **确定性强**：商业模式清晰，财务健康，管理层可靠

### 收益来源
- 企业内在价值增长（主要）
- 估值修复（次要）
- 分红再投资（辅助）

---

## 二、股票筛选流程

### 第一层：行业筛选 → 股票池A

**剔除行业**：
- 政策不确定性高或政策转向风险大的行业
- 商业模式不清晰或竞争过度激烈的行业
- 产能严重过剩行业（钢铁、煤炭、水泥等传统周期）

**保留行业**：
- 行业格局稳定、长期需求确定的行业
- 受益于产业升级、消费升级的行业
- 具备护城河的行业（技术壁垒、品牌、网络效应）

### 第二层：质量筛选 → 核心资产池（30-50只候选）

**财务指标（硬性，全部满足）**：
```yaml
ROE: ≥15%（连续5年）
毛利率: ≥40%（连续3年）
营收增长: 年均≥10%
负债率: <60%
现金流: 经营性现金流/净利润 >0.8
```

**护城河（至少2项）**：
- 品牌护城河（消费品）
- 技术护城河（专利、研发占比>8%）
- 网络效应（平台、渠道）
- 成本优势（规模经济）
- 转换成本（客户粘性高）

**行业地位**：
- 细分行业前3名
- 市占率≥15%或持续提升

**管理层**：
- 创始人/核心团队持股>5%
- 无重大违规记录
- 分红率>30%或持续回购

### 第三层：估值筛选 → 买入时机

**估值标准（增加硬约束，避免高位接盘）**：
- 触发条件（必须同时满足2个以上）：
  - **PE处于历史30分位以下**（而非50分位）
  - **PEG < 1**（市盈率增长比<1，估值合理）
  - **股息率 > 3%**（有分红保护）
  - 或：大盘系统性下跌>25%
  - 或：行业轮动被冷落，板块整体回调>30%

**核心原则**：
- 不在"优质但昂贵"时买入，等待合理估值
- 通过定投+分批建仓降低择时风险
- 重点关注企业质量，但也要等待价格合理

---

## 三、买入策略

### 建仓节奏
```
第一批: 目标仓位的40%（试探性）
第二批: 下跌10%加30%（降低成本）
第三批: 下跌20%加30%（满仓）
时间跨度: 3-6个月完成建仓
```

### 择时原则
- **不过分择时**，分批建仓降低风险
- **定投优先**（每月固定日买入）
- **大跌加速**（市场恐慌时加速建仓）

### 仓位控制
- **超级核心**（茅台、招行）：10-15%
- **普通核心**：8-12%
- **总持仓**：5-8只股票
- **行业分散**：3-5个不相关行业，单一行业≤30%

---

## 四、持仓监测

### 季度级别检查

**财报检查（季报后3天内）**：
```
□ ROE是否维持>15%
□ 营收/利润增速是否符合预期
□ 现金流是否健康
□ 毛利率是否稳定
```

**年度复盘**：
```
□ 护城河是否依然存在
□ 行业竞争格局是否恶化
□ 估值是否严重高估（PE>历史80分位）
□ 管理层是否发生重大变化
```

**调仓频率**：
- 一般不调仓，长期持有
- 每年最多调整1-2只

### 触发式复审

**复审触发条件**：
- 股价较成本下跌超20%
- 重大负面新闻
- 财报异常（ROE下滑、营收下降等）

**复审输出**：
- 逻辑完全成立 → 继续持有或加仓
- 逻辑部分成立 → 降低仓位
- 逻辑被证伪 → 执行退出

---

## 五、退出与调整

### 1. 止盈策略（谨慎使用）

**估值止盈（分批减仓）**：
```
PE达到历史80分位 → 减仓至8%
PE达到历史90分位 → 减仓至5%
涨幅>200%且估值过高 → 考虑清仓
```

**基本面止盈**：
```
ROE连续2年<12% → 清仓
护城河被摧毁 → 立即清仓
行业景气度永久逆转 → 立即清仓
```

**机会成本止盈**：
```
发现明显更优标的 → 换仓
需满足：新标的预期收益高50%+
```

### 2. 止损策略（谨慎使用）

**逻辑止损（优先级最高）**：
- 买入逻辑被证伪（如政策转向、技术路线被颠覆）
- 财务造假或治理问题暴露
- 立即清仓，不考虑价格

**价格止损**：
```
单只股票亏损达到-30%时，触发强制复审
复审后若逻辑仍成立，可继续持有或加仓
复审后若逻辑存疑，执行止损
```

**基本面止损**：
```
ROE连续2个季度下滑>30%
市场份额连续流失
主营业务恶化
```

### 3. 仓位动态调整

**加仓条件**：
- 股价下跌但基本面未恶化（更好的买入机会）
- 基本面超预期改善，确定性提升
- 定投日常规买入

**减仓条件**：
- 估值接近合理区间上限（PE>历史75分位）
- 基本面出现边际恶化迹象
- 发现更优核心资产需腾挪资金

**调仓频率**：
- 每年最多1-2次主动调仓
- 以持有为主，不频繁交易

## 八、关键原则总结

1. **质量第一**：ROE>15%、护城河深厚是铁律
2. **长期持有**：5-10年视角，忽略短期波动
3. **合理估值**：不过分择时，PE<50分位即可买入
4. **分散风险**：5-8只股票，3-5个行业
5. **纪律执行**：定投为主，季度复审，年度调整

**核心信念**：
> 时间是优质公司的朋友。通过长期持有真正的好公司，让复利发挥魔力，这是核心资产投资的精髓。
