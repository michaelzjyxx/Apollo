## 一、方案概述

### 1.1 覆盖范围

**行业范围**：申万一级行业（31个）
- 农林牧渔、采掘、化工、钢铁、有色金属、电子、家用电器、食品饮料、纺织服饰、轻工制造、医药生物、公用事业、交通运输、房地产、商贸零售、社会服务、银行、非银金融、建筑材料、建筑装饰、电力设备、国防军工、计算机、传媒、通信、汽车、机械设备、煤炭、石油石化、环保、美容护理

**指标覆盖**：23/26个指标（88%）

### 1.2 数据更新频率

| 指标类型 | 更新频率 | 操作时间 |
|---------|---------|---------|
| 宏观景气度指标 | 月度 | 每月10-15日 |
| 行业财务指标 | 季度 | 季报发布后1周内 |
| 估值指标 | 周度 | 每周一 |
| 资金流向 | 周度 | 每周五 |

### 1.3 输出成果

1. **行业数据库（Excel）**：包含所有31个行业的23项指标原始数据
2. **行业评分表（Excel）**：每个行业的各维度得分和综合得分
3. **行业池清单**：
   - 核心池（综合得分≥80分）
   - 观察池（60-80分）
   - 排除池（<60分或触发红线）

## 二、数据获取操作手册

### 2.1 前置准备

**工具准备**：
1. 同花顺iFinD客户端（已登录）
2. Excel 2016及以上版本
3. 数据整合模板（见第六章）

**申万行业代码表**：
```
在iFinD中获取：
数据浏览器 → 板块分类 → 申万一级行业 → 导出代码表

得到31个行业代码（如：801010.SI 农林牧渔、801020.SI 采掘...）
```

---

### 2.2 指标数据获取路径（分类操作）

#### 分类1：竞争格局指标（3个，iFinD可获取）

**指标1.1：行业集中度（CR5）**

```
操作路径：
iFinD → 数据浏览器 → 行业数据 → 集中度指标

具体步骤：
1. 点击"数据浏览器"
2. 左侧树形菜单选择"行业数据" → "竞争格局" → "集中度指标"
3. 选择指标：CR5（营收口径）
4. 选择行业：申万一级行业（全选31个）
5. 时间：最新报告期（如2024Q3）
6. 点击"提取数据"
7. 导出Excel：文件名"1_CR5.xlsx"

输出字段：
- 行业名称
- 行业代码
- CR5（%）
- 报告期

注意：部分细分行业可能无数据，标记为"N/A"
```

**指标1.2：龙头市场份额变化**

```
操作路径：
iFinD → 个股F10 → 行业地位 → 市场份额（手工查询）

具体步骤：
1. 先查询各行业龙头企业（按最新市值排序取第一名）
   iFinD → 数据浏览器 → 选股 → 申万一级行业
   → 按"总市值"降序排列 → 记录各行业第一名

2. 逐个查询龙头市场份额：
   个股F10 → "行业地位" → 查看"市场份额"

3. 提取近3年数据（如2022/2023/2024）

4. 手工整理到Excel：文件名"2_龙头份额.xlsx"

输出字段：
- 行业名称
- 龙头企业
- 2022年市场份额（%）
- 2023年市场份额（%）
- 2024年市场份额（%）
- 变化趋势（计算：2024-2022）

注意：部分行业龙头年报可能未披露市场份额，需查研报补充或标记"N/A"
```

**指标1.4：产能利用率**

```
操作路径：
iFinD → 宏观数据 → 工业数据 → 产能利用率

具体步骤：
1. 宏观数据 → 中国 → 工业 → 产能利用率
2. 选择行业：匹配申万行业（注意口径差异）
3. 时间：最新季度
4. 导出Excel：文件名"3_产能利用率.xlsx"

输出字段：
- 行业名称
- 产能利用率（%）
- 报告期

注意：
- iFinD产能利用率覆盖约15个重点行业（钢铁、水泥、化工等）
- 其他行业标记"N/A"或查行业协会数据补充
```

#### 分类4：现金流指标（2个）

**指标4.1：经营性现金流/净利润**

```
操作路径：
iFinD → 行业数据 → 现金流指标

具体步骤：
1. 数据浏览器 → 行业数据 → 现金流量表
2. 自定义指标：经营活动现金流净额 / 净利润
   （iFinD支持公式自定义）
3. 选择行业：申万一级行业（全选）
4. 时间：最新报告期（TTM）
5. 导出Excel：文件名"8_现金流质量.xlsx"

- 行业名称
- 经营现金流/净利润（%）
- 报告期

公式设置：
iFinD公式编辑器：
CASHFLOW_OP / NET_PROFIT
（具体字段代码见iFinD帮助文档）
```

**指标4.2：资本支出强度**

```
操作路径：
iFinD → 行业数据 → 现金流量表 → Capex、折旧

具体步骤：
1. 数据浏览器 → 行业数据 → 现金流量表
2. 提取两个指标：
   a) 购建固定资产支付的现金（Capex）
   b) 固定资产折旧（从利润表附注获取）
3. 选择行业：申万一级行业（全选）
4. 时间：最新年度（年度数据更准确）
5. 导出Excel：文件名"9_资本支出.xlsx"

输出字段：
- 行业名称
- Capex（亿元）
- 折旧（亿元）
- Capex/折旧（自己计算）
- 报告期

注意：iFinD无直接比值，需导出后在Excel中计算
```

#### 分类5：估值指标（2个）

**指标5.1：行业PE/PB历史分位**

```
操作路径：
iFinD → 行业估值 → 申万行业 → PE/PB Band

具体步骤：
1. 行情 → 板块 → 申万行业
2. 右键点击某个行业 → "估值分析" → "PE Band"
3. 选择时间区间：近5年
4. 记录当前PE分位数（显示在图表上）
5. 重复操作31个行业（较繁琐）

或使用批量方式：
1. 数据浏览器 → 行业估值指标
2. 选择指标：PE-TTM、PB、当前值、5年分位数
3. 选择行业：申万一级行业（全选）
4. 导出Excel：文件名"10_估值分位.xlsx"

输出字段：
- 行业名称
- PE-TTM（倍）
- PE 5年分位（%）
- PB（倍）
- PB 5年分位（%）
- 报告日期
```

**指标5.2：PEG**

```
数据获取：
需要两个数据：PE + 利润增速预期

步骤1：获取PE
iFinD → 行业估值 → PE-TTM（已在上一步获取）

步骤2：获取利润增速（一致预期）
路径：iFinD → 一致预期 → 行业盈利预测 → 净利润增速

具体操作：
1. 数据浏览器 → 一致预期 → 行业盈利预测
2. 选择指标：未来1年净利润增速（%）
3. 选择行业：申万一级行业
4. 导出Excel：文件名"11_盈利预期.xlsx"

输出字段：
- 行业名称
- PE-TTM（倍）
- 一致预期增速（%）
- PEG（计算：PE/增速）

注意：
- 部分行业可能无一致预期覆盖，用近3年CAGR替代
- 周期行业PEG失真，可标记"N/A"
```

#### 分类9：红线触发指标（4个）

导出：并入"6_营收增速.xlsx"

IF(连续2季度同比<-20%, "触发红线", "正常")

```
数据来源：指标5.1已获取PE/PB分位

判断：
IF(PE分位>95% OR PB分位>95%, "触发红线", "正常")
```

```

### 2.3 缺失指标的替代方案

**缺失1：价格体系稳定性（工业品现货价格）**

```
iFinD替代方案：
1. 部分期货价格可获取
   路径：商品 → 期货价格 → 主力合约

2. 主要依赖外部数据源：
   - 卓创资讯（部分免费）
   - 生意社（部分免费）

建议：
- 优先监控周期品行业（钢铁、化工、煤炭等）
- 每月手工更新到Excel
- 计算近12月价格波动率（CV）
```

```
iFinD辅助工具：

## 三、数据整合与计算

### 3.1 数据整合架构

**目标**：将21个Excel文件整合到1个主数据表

**主数据表结构**：`行业筛选数据库.xlsx`

### 3.2 关键计算公式（Excel函数）

**3.2.1 历史分位计算**

```excel
说明：计算当前值在历史序列中的百分位

示例：ROE历史分位

假设：
- 当前ROE在单元格 B2
- 历史5年ROE序列在 C2:V2（20个季度）

公式：
=PERCENTRANK(C2:V2, B2, 2) * 100

结果：返回0-100的百分位数
```

**3.2.2 毛利率趋势判断**

```excel
说明：判断近8季度毛利率趋势（上升/平稳/下降）

假设：
- 近8季度毛利率在 G2:N2

公式：
=IF(SLOPE(G2:N2, {1;2;3;4;5;6;7;8})>0.5, "上升",
  IF(SLOPE(G2:N2, {1;2;3;4;5;6;7;8})<-0.5, "下降", "平稳"))

说明：
- SLOPE计算线性趋势斜率
- >0.5：上升趋势
- <-0.5：下降趋势
```

**3.2.3 库存周期四象限判断**

```excel
说明：根据存货同比和营收同比判断库存周期位置

假设：
- 存货同比在 P2
- 营收同比在 Q2

公式：
=IF(AND(P2>5%, Q2>10%), "被动补库（景气上行）",
  IF(AND(P2<-5%, Q2<0%), "主动去库（景气底部）",
    IF(AND(P2>10%, Q2<5%), "主动补库（景气高位）",
      IF(AND(P2<0%, Q2<-5%), "被动去库（景气下行）", "过渡期"))))

输出：
- "被动补库（景气上行）" → 优选
- "主动去库（景气底部）" → 关注拐点
- "主动补库（景气高位）" → 警惕
- "被动去库（景气下行）" → 回避
```

**3.2.4 红线触发判断**

```excel
1. 毛利率红线
假设：近8季度毛利率在 G2:N2

公式：
=IF(AND(N2<M2, M2<L2, L2<K2, K2<J2, (J2-N2)>5), "触发", "正常")

说明：连续4季度下滑且累计降幅>5pct

2. 估值红线
假设：PE分位在 S2, PB分位在 U2

公式：
=IF(OR(S2>95, U2>95), "触发", "正常")

3. 资金红线
假设：8周累计资金流入在 Z2

公式：
=IF(Z2<-100, "触发", "正常")
```

### 4.4 景气度调整（±5分）

**调整逻辑**：根据宏观景气度指标对周期品行业进行加减分

**调整对象**：
- 强周期行业：钢铁、煤炭、有色金属、化工、建材
- 中周期行业：机械、电力设备、建筑
- 弱周期行业：其他

**调整标准**：

| 宏观指标组合 | 周期品调整 | 说明 |
|------------|-----------|------|
| PMI>52 且 PPI>3% 且 社融>10% | +5 | 强景气 |
| PMI>50 且 PPI>0% 且 社融>8% | +3 | 温和景气 |
| PMI 48-50 | 0 | 中性 |
| PMI<48 且 PPI<-2% | -3 | 弱景气 |
| PMI<45 且 PPI<-5% | -5 | 严重衰退 |

**Excel公式**：
```excel
景气度调整 =
IF(行业类型<>"周期", 0,
  IF(AND(PMI>52, PPI>3, 社融>10), 5,
    IF(AND(PMI>50, PPI>0, 社融>8), 3,
      IF(AND(PMI<48, PPI<-2), -3,
        IF(AND(PMI<45, PPI<-5), -5, 0)))))
```

---

### 4.5 红线扣分（-10分）

**扣分规则**：触发任一红线，扣10分

| 红线指标 | 触发条件 | 扣分 |
|---------|---------|------|
| 毛利率连续下滑 | 连续4季度下滑且累计>5pct | -10 |
| 营收连续下滑 | 连续2季度同比<-20% | -10 |
| 估值历史高位 | PE或PB分位>95% | -10 |
| 资金持续流出 | 连续8周净流出且累计<-100亿 | -10 |

**Excel公式**：
```excel
红线扣分 =
IF(OR(毛利率红线="触发", 营收红线="触发", 估值红线="触发", 资金红线="触发"), -10, 0)
```

**注意**：
- 红线是一票否决机制
- 即使其他维度得分高，触发红线后总分大幅降低
- 触发红线的行业直接进入"排除池"

### 4.6 综合得分计算

**最终得分公式**：

```
综合得分 =
  竞争格局得分（0-15）
+ 盈利能力得分（0-15）
+ 成长性得分（0-15）
+ 现金流得分（0-10）
+ 估值得分（0-10）
+ 周期位置得分（0-5）
+ 政策环境得分（0-5）
+ 商业模式得分（0-5）
+ 进入壁垒得分（0-5）
+ 护城河得分（0-5）
+ 景气度调整（-5~+5）
+ 红线扣分（0或-10）

理论最高分：100 + 5 = 105
理论最低分：0 - 5 - 10 = -15（实际取0）
```

**Excel总公式**：
```excel
=MAX(0,
  竞争格局总分 + 盈利能力总分 + 成长性总分 + 现金流总分 + 估值总分 +
  周期位置分 + 政策评分 + 商业模式分 + 进入壁垒分 + 护城河分 +
  景气度调整 + 红线扣分)
```

### 4.7 行业池分类

**分类标准**：

| 行业池 | 综合得分 | 其他条件 | 操作建议 |
|-------|---------|---------|---------|
| **核心池** | ≥80分 | 未触发红线 | 重点配置，可深入个股筛选 |

## 八、常见问题FAQ

**Q1：iFinD没有某些行业的数据怎么办？**

A：按优先级处理：
1. 查看是否是数据更新延迟，等几天再查
2. 尝试用二级行业数据加权计算
3. 查外部免费数据源（统计局、行业协会）
4. 标记为"N/A"，按中性给分

---

**Q2：31个行业太多，能否精简？**

A：可以。建议先筛选：
1. 剔除明确不投资的行业（如银行、地产、建筑）
2. 聚焦15-20个行业即可
3. 但建议保留完整数据库，方便未来扩展

---

**Q3：评分权重如何调整？**

A：根据投资风格：
- 价值风格：提高"盈利能力"和"估值"权重
- 成长风格：提高"成长性"权重
- 周期风格：提高"周期位置"和"景气度调整"权重

在"参数设置"表中修改即可

---

**Q4：手工数据（如龙头份额）太费时间，能否简化？**

A：
1. 降低更新频率（如半年更新一次）
2. 或用研报数据替代（券商行业深度报告通常有）
3. 或该指标直接标记"N/A"，权重分配给其他指标

---

**Q5：红线标准是否太严？**

A：
- 红线标准可在"参数设置"中调整
- 建议先按默认值运行1-2个季度
- 根据实际情况再调整（如PE分位从95%改为90%）
