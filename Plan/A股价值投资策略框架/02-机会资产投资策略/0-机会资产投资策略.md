# 机会资产投资策略

**定位**：占总仓位30%，捕捉优质股错杀机会
**目标**：通过1-2年持有期，获得单次40-60%收益（年化20-25%）
**核心理念**：在市场恐慌时买入临时困境的优质公司，等待估值修复

---

## 一、机会资产定义

### 资产特征
- **基本面健康**：ROE≥12%，财务无重大风险
- **短期困境**：受市场情绪、行业周期、突发事件影响被错杀
- **深度低估**：PE<历史20分位 或 PB<1.2
- **明确催化剂**：政策转向、周期反转、事件平息等可预期的修复因素

### 收益来源
- 估值修复（主要）：从极度低估回归合理估值
- 业绩改善（次要）：困境反转带来的盈利增长
- 情绪修复（辅助）：市场恐慌消退后的资金回流

### 与核心资产的区别
- **持有周期短**：1-2年 vs 5-10年
- **标准更宽松**：ROE≥12% vs ≥15%
- **强调择时**：必须等到深度低估 vs 合理估值即可
- **主动退出**：涨幅达标即卖 vs 长期持有

---

## 二、股票筛选流程

### 第一层：行业筛选（与核心资产共用）

**剔除行业**：
- 政策不确定性高或政策转向风险大的行业
- 商业模式不清晰或竞争过度激烈的行业
- 产能严重过剩行业

**保留行业**：
- 行业格局稳定、长期需求确定的行业
- 受益于产业升级、消费升级的行业
- 具备护城河的行业

### 第二层：质量筛选 → 机会资产池（20-30只候选）

**财务指标（满足基础即可）**：
```yaml
ROE: ≥12%（允许短期波动）
行业地位: 细分前5
负债率: <70%
现金流: 无债务违约风险
```

**护城河**：
- 至少有1个明显护城河
- 竞争格局稳定

**排除项**：
```
❌ ST、*ST股票
❌ 商誉占净资产>30%
❌ 连续2年业绩下滑>20%
❌ 财务造假历史
```

### 第三层：估值筛选 → 买入时机（核心环节）

**严格等待错杀，必须深度低估**

**触发条件（必须同时满足3个）**：
1. **估值极低**：PE<历史20分位 或 PB<1.2
2. **深度回撤**：从高点回撤>35%
3. **明确催化剂**：
   - 政策转向（如地产政策放松）
   - 周期反转（如猪周期见底）
   - 事件平息（如食品安全事件过去）
   - 业绩拐点（如新产品放量）

**错杀类型识别**：
- **情绪错杀**：市场恐慌导致的无差别抛售
- **周期错杀**：行业周期底部的超跌
- **事件错杀**：负面事件冲击后的过度反应
- **政策错杀**：政策收紧导致的板块暴跌

---

## 三、买入策略

### 建仓节奏（快速建仓）
```
第一批: 下跌35%买入50%
第二批: 下跌45%买入30%
第三批: 下跌55%买入20%
时间跨度: 1-3个月快速建仓
```

### 择时原则
- **严格择时**，等待恐慌时刻
- **必须满足深度低估条件**（PE<20分位 + 回撤>35%）
- **需要明确的催化剂**（不买没有催化剂的标的）

### 仓位控制
- **单只上限**：5-8%（严格控制）
- **总持仓**：3-5只
- **避免同行业集中**：降低行业系统性风险
- **不集中持仓**：因持有周期短，不确定性高

## 四、持仓监测

### 周/月/季度级别检查


### 触发式复审

**复审触发条件**：
- 亏损达到-25%（强制复审）
- 催化剂落空或延期
- 基本面连续2季度不及预期

**复审输出**：
- 逻辑成立 → 继续持有或加仓
- 逻辑被证伪 → 立即止损

## 五、退出与调整

### 1. 止盈策略（核心策略）

**目标价止盈（主要退出方式）**：
```
涨幅>50% 且 PE回到历史中位数 → 分批卖出
涨幅>80% → 直接卖出，不贪心
```

**分批卖出计划**：
```
涨幅达50%: 卖出30%（锁定部分利润）
涨幅达70%: 卖出40%（保留少量仓位）
涨幅达90%: 全部清仓（获利了结）
```

**时间止盈**：
```
持有满18个月，涨幅>30% → 分批卖出
持有满24个月，无论盈亏 → 重新评估是否继续持有
```

**催化剂兑现止盈**：
```
买入时的催化剂已完全兑现 → 卖出
估值修复到位（PE回到中位数） → 卖出
```

**机会成本止盈**：
```
发现更好的错杀标的（预期收益差>30%） → 换仓
```

### 2. 止损策略（严格执行）

**逻辑止损（优先级最高）**：
- 买入逻辑被证伪 → 立即止损
- 催化剂彻底落空 → 立即止损
- 财务造假或治理问题暴露 → 立即清仓

**时间止损**：
```
持有12个月仍亏损>15% → 清仓
持有24个月仍未达到预期 → 清仓
```

**催化剂止损**：
```
预期的催化剂落空 → 清仓
基本面连续2季度不及预期 → 清仓
```

**价格止损**：
```
亏损达到-25% → 强制复审
复审后逻辑不成立 → 立即止损
```

### 3. 仓位动态调整

**加仓条件**：
```
继续下跌且催化剂更明确
按计划分批建仓（第二批、第三批）
```

**减仓/清仓条件**：
```
达到目标价（主要退出方式）
时间到期（18-24个月）
发现更好机会标的
催化剂落空或基本面恶化
```

**调仓频率**：
- 相对频繁（年均1-2次完整周期）
- 达到目标即退出，寻找下一个机会
- 不恋战，不贪心

## 八、关键原则总结

1. **严格择时**：必须等到深度低估（PE<20分位 + 回撤>35%）
2. **明确催化剂**：不买没有催化剂的标的
3. **快速建仓**：1-3个月内完成建仓
4. **主动退出**：涨幅达标即卖，不贪心
5. **纪律止损**：逻辑被证伪立即止损
6. **高频监控**：每周检查，及时调整

**核心信念**：
> 市场先生经常情绪化地给出不合理的报价。机会资产投资的精髓，是在恐慌时勇敢买入，在贪婪时果断卖出。

## 九、常见错杀场景案例

### 1. 政策错杀
**案例**：2021年教育双减政策 → 教育股暴跌
**特征**：政策一刀切，市场恐慌性抛售
**机会**：职业教育、素质教育等不受影响的细分领域被错杀
**催化剂**：政策明确边界，市场情绪平复

### 2. 周期错杀
**案例**：2022年猪周期底部 → 生猪养殖股暴跌
**特征**：行业周期底部，全行业亏损
**机会**：龙头企业具备成本优势，能熬过周期
**催化剂**：产能去化，猪价反转

### 3. 事件错杀
**案例**：2008年三聚氰胺事件 → 乳制品股暴跌
**特征**：行业性负面事件，市场无差别抛售
**机会**：未涉事的龙头企业被错杀
**催化剂**：事件平息，市占率反而提升

### 4. 情绪错杀
**案例**：2018年中美贸易战 → 科技股暴跌
**特征**：市场恐慌情绪蔓延，优质公司被错杀
**机会**：基本面未受影响的内需型科技股
**催化剂**：贸易谈判进展，情绪修复

## 十、风险提示

### 需要警惕的"伪错杀"
- **基本面恶化伪装的错杀**：业绩持续下滑，不是短期困境
- **行业永久衰退伪装的周期**：如传统媒体、功能手机
- **估值陷阱**：PE很低但ROE持续下降
- **催化剂永不兑现**：如某些"重组概念股"

### 控制风险的关键
1. **严格执行30%仓位上限**：机会资产风险高于核心资产
2. **单只不超8%**：避免单一标的风险过大
3. **止损纪律**：亏损25%或持有12个月未见起色，果断止损
4. **分散持仓**：同时持有3-5只，避免押注单一机会
