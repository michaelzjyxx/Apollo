# 第三层：估值筛选 → 买入时机（核心环节）

**严格等待错杀，必须深度低估**

**触发条件（必须同时满足3个）**：
1. **估值极低**：PE<历史20分位 或 PB<1.2
2. **深度回撤**：从高点回撤>35%
3. **明确催化剂**：
   - 政策转向（如地产政策放松）
   - 周期反转（如猪周期见底）
   - 事件平息（如食品安全事件过去）
   - 业绩拐点（如新产品放量）

**错杀类型识别**：
- **情绪错杀**：市场恐慌导致的无差别抛售
- **周期错杀**：行业周期底部的超跌
- **事件错杀**：负面事件冲击后的过度反应
- **政策错杀**：政策收紧导致的板块暴跌
