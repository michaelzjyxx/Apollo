# 二、股票筛选流程

三层筛选流程，从行业到个股，逐步缩小范围。

## 筛选架构

```
┌──────────┐   ┌──────────┐   ┌──────────┐
│ 第一层   │   │ 第二层   │   │ 第三层   │
│ 行业筛选 │──►│ 质量筛选 │──►│ 择时策略 │
└──────────┘   └──────────┘   └──────────┘
     │              │              │
     ▼              ▼              ▼
 优质行业        优质公司        买入信号
 (20-30个)      (50-100只)      (动态)
 季度更新        季度更新       实时/日更新
```

## 各层职责

| 层级 | 目标 | 核心问题 | 更新频率 |
|------|------|----------|----------|
| 第一层 | 行业筛选 | 这个行业值得投资吗？ | 季度 |
| 第二层 | 质量筛选 | 这家公司基本面好吗？ | 季度 |
| 第三层 | 择时策略 | 这家公司现在被错杀了吗？ | 日/实时 |

## 子章节

- [第一层：行业筛选](./1：行业筛选.md) - 与核心资产共用的行业筛选标准
- [第二层：质量筛选](./2：质量筛选.md) - 从优质行业中筛选出基本面优秀的公司（50-100只）
- [第三层：择时策略](./3：择时策略.md) - 在优质公司中寻找被错杀的机会，生成买入信号
