## 一、竞争格局指标

### 1.1 行业集中度（CR5）

**指标定义**：行业前5名企业的市场份额合计

**数据来源**：
| 平台 | 获取路径 | 说明 |
|-----|---------|------|
| **Wind** | 行业 → 行业分析 → 市场集中度 → CR5 | 覆盖全面，数据准确 |
| **同花顺iFinD** | 数据浏览器 → 行业数据 → 集中度指标 | 可自定义计算 |
| **Choice** | 行业研究 → 行业对比 → 市场份额 | 数据较新 |

**手工计算方法**（无数据终端时）：
```
步骤：
1. 在东方财富/同花顺查询行业所有上市公司
2. 按营收/销量排序，取前5名
3. CR5 = (前5名营收合计 / 全行业营收) × 100%
```

**判断标准**：
- CR5 > 60%：高度集中（寡头垄断）
- CR5 40%-60%：较为集中（优选范围）
- CR5 20%-40%：适度集中
- CR5 < 20%：高度分散（剔除）

**注意事项**：
- 零售、餐饮等天然分散行业，应采用区域CR5或细分品类CR5
- 统计口径需一致（如都用营收，或都用销量）

### 1.2 龙头市场份额变化

**指标定义**：行业第一名企业的市场份额及其变化趋势

**数据来源**：
| 平台 | 获取路径 |
|-----|---------|
| **Wind** | 个股 → 公司研究 → 市场地位 → 市场份额 |
| **同花顺** | F10 → 行业地位 → 市场份额 |
| **年报/公告** | 上市公司年报"业务概要"章节 |

**分析方法**：
```
1. 提取近3年龙头企业市场份额数据
2. 计算年度变化率
3. 判断趋势：
   - 持续提升（+0.5pct/年以上）：格局优化 ✓
   - 基本稳定（±0.3pct以内）：格局稳定 ✓
   - 持续下降（-0.5pct/年以上）：竞争加剧 ✗
```

**主观判断要素**：
- 查看行业研报，了解份额变化原因（产品创新？价格战？）
- 关注龙头二、三名的动向（是否有追赶趋势）

---

### 1.3 价格体系稳定性

**指标定义**：行业产品价格波动情况

**数据来源**：
| 数据类型 | 获取渠道 | 更新频率 |
|---------|---------|---------|
| **工业品价格** | 卓创资讯、生意社、百川盈孚、钢联数据 | 日/周 |
| **消费品价格** | 国家统计局CPI分项、中商产业研究院 | 月 |
| **大宗商品** | Wind商品价格、文华财经 | 日 |

**具体查询**：
- **卓创资讯**：www.sci99.com → 价格中心 → 选择品种
- **生意社**：www.100ppi.com → 价格指数
- **统计局CPI分项**：stats.gov.cn → 月度数据 → 居民消费价格指数

**判断标准**：
```
计算价格波动率（CV）= 标准差 / 均值
- CV < 10%：价格稳定 ✓
- CV 10%-20%：适度波动
- CV > 20%：价格剧烈波动 ✗
```

**主观判断**：
- 是否频繁发生价格战（查行业新闻、研报）
- 龙头企业是否具备定价权（如茅台、片仔癀）

### 1.4 产能利用率

**指标定义**：实际产量占设计产能的比例

**数据来源**：
| 来源 | 获取路径 | 说明 |
|-----|---------|------|
| **国家统计局** | 工业统计 → 产能利用率 | 仅部分重点行业 |
| **行业协会** | 各行业协会官网/年度报告 | 如中国钢铁工业协会、水泥协会等 |
| **Wind** | 行业 → 行业数据 → 产能利用率 | 部分行业有数据 |
| **研报** | 券商研报"供需分析"章节 | 通常有测算 |

**重点行业数据源**：
| 行业 | 数据来源 |
|-----|---------|
| 钢铁 | 中国钢铁工业协会 www.chinaisa.org.cn |
| 水泥 | 中国水泥协会 www.zgssnl.com |
| 化工 | 隆众资讯、卓创资讯 |
| 煤炭 | 国家煤矿安监局、煤炭工业协会 |

**无直接数据时的推算方法**：
```
产能利用率 = 实际产量 / 设计产能

其中：
- 实际产量：统计局公布的行业产量数据
- 设计产能：上市公司公告产能合计 + 行业协会统计
```

**判断标准**：
- 产能利用率 > 85%：供需偏紧 ✓
- 产能利用率 70%-85%：供需平衡
- 产能利用率 < 70%：产能过剩 ✗

## 二、盈利能力指标

### 2.1 行业平均ROE

**指标定义**：行业上市公司净资产收益率的平均值

**数据来源**：
| 平台 | 获取路径 | 计算方式 |
|-----|---------|---------|
| **Wind** | 行业 → 行业对比 → 财务指标 → ROE（TTM） | 市值加权平均 |
| **理杏仁** | 行业估值 → 申万行业 → ROE | 等权平均 |
| **同花顺iFinD** | 数据浏览器 → 行业财务指标 | 可选加权方式 |

**手工计算**（无终端时）：
```
1. 筛选行业内所有A股上市公司
2. 剔除ST、亏损、上市不满1年的公司
3. 提取每家公司的ROE（TTM）
4. 计算：
   - 简单平均：Σ(ROE) / 公司数量
   - 市值加权：Σ(ROE × 市值) / Σ市值（推荐）
```

**判断标准（相对比较法）**：
| 对比维度 | 优选标准 | 数据来源 |
|---------|---------|---------|
| 横向对比 | 高于全A中位数（约8%） | Wind行业对比 |
| 纵向对比 | 高于自身5年历史50%分位 | 理杏仁历史分位 |
| 行业特性 | 制造业>12%，消费>18%，周期>10% | 行业经验值 |

**历史分位查询**：
- **理杏仁**：lixinger.com → 指数估值 → 申万行业 → ROE → 查看历史走势
- **Wind**：行业分析 → 历史数据 → 导出后自行计算分位

---

### 2.2 行业平均毛利率

**指标定义**：行业上市公司毛利率的平均值

**数据来源**：
| 平台 | 获取路径 |
|-----|---------|
| **Wind** | 行业 → 行业对比 → 毛利率（TTM） |
| **同花顺** | 行业数据 → 盈利能力 → 毛利率 |
| **Choice** | 行业财务指标 → 毛利率 |

**计算公式**：
```
行业平均毛利率 = Σ(单个公司毛利率 × 营收) / Σ营收
```

**判断标准**：
- **绝对值标准**（仅供参考）：
  - 高毛利行业（>50%）：软件、白酒、医药
  - 中毛利行业（20%-50%）：家电、电子、食品
  - 低毛利行业（<20%）：零售、钢铁、水泥

- **相对标准**（推荐）：
  - 高于自身5年均值，且环比改善 ✓
  - 连续4个季度下滑 >5pct ✗

**趋势分析**：
```
在Wind/同花顺中：
1. 提取近3年季度毛利率数据
2. 绘制趋势图
3. 判断：上行✓、平稳✓、下行✗
```

## 三、成长性指标

### 3.1 行业收入增速

**指标定义**：行业上市公司营业收入同比增长率

**数据来源**：
| 平台 | 获取路径 | 统计口径 |
|-----|---------|---------|
| **Wind** | 行业 → 行业对比 → 营业收入增长率（TTM） | 市值加权 |
| **同花顺** | 行业数据 → 成长能力 → 营收增速 | 等权平均 |
| **国家统计局** | 工业统计 → 工业增加值增速 | 规模以上企业 |

**多口径对比**：
| 数据口径 | 优点 | 缺点 | 适用场景 |
|---------|-----|------|--------|
| 上市公司TTM | 及时性强 | 样本偏差 | 快速判断 |
| 工业增加值 | 覆盖全面 | 滞后1-2月 | 制造业行业 |
| 行业协会数据 | 权威性高 | 更新慢 | 年度复盘 |

**判断标准（相对比较法）**：
```
优选标准：
1. 行业收入增速 > GDP增速 + 3%（如2024年GDP 5%，则>8%）
2. 或：处于自身历史70%分位以上（查理杏仁）
3. 且：连续3个季度正增长
```

**统计局数据查询**：
```
stats.gov.cn → 月度数据 → 工业 → 工业增加值增速
→ 选择细分行业（如"医药制造业"）
```

---

### 3.2 行业利润增速

**指标定义**：行业上市公司净利润同比增长率

**数据来源**：
| 平台 | 获取路径 |
|-----|---------|
| **Wind** | 行业 → 行业对比 → 净利润增长率（TTM） |
| **同花顺** | 行业数据 → 成长能力 → 净利润增速 |
| **统计局** | 工业企业利润增速（规上企业） |

**判断标准**：
```
利润弹性 = 利润增速 / 收入增速

- 利润弹性 > 1.2：盈利能力改善（规模效应、降本增效）✓
- 利润弹性 0.8-1.2：利润与收入同步
- 利润弹性 < 0.8：盈利能力恶化（竞争加剧、成本上升）✗
```

**统计局数据**：
```
stats.gov.cn → 月度数据 → 工业 → 工业企业利润
→ 按行业分类查看
```

## 四、现金流指标

### 4.1 经营性现金流/净利润

**指标定义**：经营活动现金流净额与净利润的比值

**数据来源**：
| 平台 | 获取路径 |
|-----|---------|
| **Wind** | 行业 → 行业对比 → 现金流量表 → 经营现金流/净利润 |
| **同花顺** | 财务指标 → 现金流指标 → 现金流净额/净利润 |

**手工计算**：
```
比值 = 经营活动现金流净额 / 净利润

数据提取：
1. Wind: 公司 → 财务报表 → 现金流量表 → 经营活动现金流净额
2. Wind: 公司 → 财务报表 → 利润表 → 净利润
3. 计算行业所有公司的加总比值
```

**判断标准**：
```
- 比值 > 100%：现金流优于利润（优质）✓✓
- 比值 80%-100%：现金流健康 ✓
- 比值 50%-80%：需关注应收账款、存货
- 比值 < 50%：现金流质量差 ✗

且需满足：连续3年为正
```

**异常值排查**：
- 比值异常高（>150%）：查看是否有大额资产处置
- 比值为负：企业亏损或现金流极差

---

### 4.2 资本支出强度

**指标定义**：资本支出（Capex）与折旧的比值

**数据来源**：
| 数据项 | 获取路径 |
|-------|---------|
| **资本支出** | Wind: 现金流量表 → 购建固定资产、无形资产和其他长期资产支付的现金 |
| **折旧摊销** | Wind: 现金流量表 → 固定资产折旧、油气资产折耗、生产性生物资产折旧 |

**计算公式**：
```
资本支出强度 = 资本支出(Capex) / 折旧摊销(D&A)
```

**判断标准**：
```
- 比值 < 1：维持性投资，产能收缩
- 比值 1-2：合理扩张，产能稳定增长 ✓
- 比值 2-3：快速扩张，需警惕产能过剩
- 比值 > 3：激进扩张 ✗
```

**行业差异**：
| 行业类型 | 合理区间 | 说明 |
|---------|---------|------|
| 成熟行业 | 0.8-1.5 | 维持现有产能 |
| 成长行业 | 1.5-2.5 | 适度扩张 |
| 轻资产行业 | < 1 | 低资本投入 |

## 五、估值指标

### 5.1 行业PE/PB历史分位

**指标定义**：当前估值在过去5年中的相对位置

**数据来源**：
| 平台 | 获取路径 | 说明 |
|-----|---------|------|
| **理杏仁** | 指数估值 → 申万行业指数 → PE/PB历史分位 | 免费，可视化强 |
| **Wind** | 行业 → 估值分析 → PE/PB Band → 历史分位 | 数据全，需付费 |
| **韭圈儿** | 行业估值 → 申万指数 → 估值分位 | 免费，数据较新 |

**具体查询步骤（理杏仁）**：
```
1. 访问 lixinger.com
2. 指数估值 → 申万行业
3. 选择目标行业（如"食品饮料"）
4. 查看PE-TTM、PB的"当前分位"（过去5年）
```

**判断标准**：
```
- PE/PB分位 < 30%：低估区域 ✓✓
- PE/PB分位 30%-50%：合理区域 ✓
- PE/PB分位 50%-70%：略贵
- PE/PB分位 > 70%：高估区域 ✗
```

**注意事项**：
- 周期性行业（煤炭、钢铁）优先看PB
- 成长性行业（科技、医药）优先看PE
- 金融行业（银行、保险）优先看PB

---

### 5.2 PEG（市盈率相对盈利增速比）

**指标定义**：PE / 净利润增长率

**数据来源**：
| 数据项 | 获取路径 |
|-------|---------|
| **PE** | Wind/理杏仁 → 行业PE-TTM |
| **增长率** | Wind → 行业利润增速（一致预期） |

**计算公式**：
```
PEG = PE / 净利润增长率(%)

示例：
某行业PE=30倍，预期利润增速25%
PEG = 30 / 25 = 1.2
```

**判断标准**：
```
- PEG < 1：低估 ✓✓
- PEG 1-1.5：合理 ✓
- PEG 1.5-2：略贵
- PEG > 2：高估 ✗
```

**数据获取难点**：
- **增长率**：优先用一致预期（Wind: 行业 → 一致预期 → 净利润增速）
- 无一致预期时：用近3年复合增长率（CAGR）

**适用范围**：
- 适用于成长股、成长行业
- 不适用于周期股（利润波动大）、亏损企业

## 六、景气度领先指标

### 6.1 制造业PMI

**指标定义**：采购经理人指数，50为荣枯线

**数据来源**：
| 来源 | 发布时间 | 获取路径 |
|-----|---------|---------|
| **国家统计局** | 每月1日 | stats.gov.cn → 月度数据 → PMI |
| **财新PMI** | 每月初 | 财新网、Wind |
| **Wind** | 实时更新 | 宏观 → 景气指数 → 制造业PMI |

**直接访问**：
```
国家统计局: http://www.stats.gov.cn/sj/zxfb/202501/t20250101_xxxx.html
（每月1号上午9:00发布）
```

**细分指数**：
| 分项指数 | 含义 | 领先性 |
|---------|-----|--------|
| 新订单指数 | 未来需求 | 领先2-3月 |
| 生产指数 | 当前产出 | 同步 |
| 原材料库存 | 补库/去库 | 领先1-2月 |
| 从业人员 | 就业景气 | 滞后 |

**判断标准**：
```
景气判断：
- PMI > 50 且连续3月上行 → 景气上行期 ✓
- PMI > 52 且新订单>52 → 景气高位期
- PMI < 50 且连续3月下行 → 景气下行期 ✗
- PMI < 48 且企稳 → 景气底部期（关注拐点）
```

---

### 6.2 新订单指数

**指标定义**：PMI分项，反映未来需求

**数据来源**：
```
国家统计局PMI报告 → 分类指数 → 新订单指数
Wind → 宏观 → PMI → 新订单指数
```

**判断标准**：
```
- 新订单指数 > 51 且上升 → 需求扩张 ✓
- 新订单指数 50-51 → 需求平稳
- 新订单指数 < 50 → 需求收缩 ✗

关键信号：
- 新订单 - 产成品库存 > 3 → 强补库周期
```

---

### 6.3 社融/M2增速

**指标定义**：社会融资规模/货币供应量增速

**数据来源**：
| 数据 | 发布机构 | 发布时间 | 获取路径 |
|-----|---------|---------|---------|
| **社融** | 中国人民银行 | 每月10-15日 | pbc.gov.cn → 统计数据 |
| **M2** | 中国人民银行 | 每月10-15日 | 同上 |
| **Wind** | - | 实时 | 宏观 → 货币金融 → M2/社融 |

**直接访问**：
```
央行官网: http://www.pbc.gov.cn/diaochatongjisi/116219/index.html
→ 金融统计数据报告
```

**判断标准（领先3-6个月）**：
```
金融周期判断：
- M2/社融同比 > 10% → 流动性宽松，利好股市 ✓
- M2/社融同比 8%-10% → 流动性适度
- M2/社融同比 < 8% → 流动性收紧 ✗

行业适用：
- 金融、地产：高度相关（领先6个月）
- 基建、建材：中度相关（领先3-6个月）
- 消费、医药：弱相关
```

### 6.4 PPI/CPI同比

**指标定义**：
- PPI：工业品出厂价格指数（影响周期品）
- CPI：居民消费价格指数（影响消费品）

**数据来源**：
| 指数 | 发布时间 | 获取路径 |
|-----|---------|---------|
| **PPI** | 每月10日左右 | stats.gov.cn → 月度数据 → PPI |
| **CPI** | 每月10日左右 | stats.gov.cn → 月度数据 → CPI |
| **Wind** | 实时 | 宏观 → 物价 → PPI/CPI |

**细分项获取**：
```
统计局官网 → 数据 → 价格指数 → PPI/CPI分类
→ 例如：CPI-食品、PPI-黑色金属
```

**判断标准**：
```
周期行业（看PPI）：
- PPI同比 > 5% → 价格强势，利好周期品 ✓
- PPI同比 -2% ~ 2% → 价格平稳
- PPI同比 < -2% → 通缩风险 ✗

消费行业（看CPI）：
- CPI同比 2%-3% → 温和通胀，利好消费 ✓
- CPI同比 > 3% → 通胀压力，关注成本传导
- CPI同比 < 1% → 需求疲软 ✗
```

## 七、周期位置判断指标

### 7.1 存货同比

**指标定义**：上市公司存货余额的同比增长率

**数据来源**：
| 平台 | 获取路径 |
|-----|---------|
| **Wind** | 行业 → 行业对比 → 资产负债表 → 存货 → 同比增速 |
| **同花顺** | 行业数据 → 资产负债表 → 存货同比 |

**计算方法**：
```
存货同比 = (本期存货 - 去年同期存货) / 去年同期存货 × 100%
```

**库存周期判断**（结合营收同比）：
```
              营收↑        营收↓
存货↓    主动去库存    被动去库存
         （底部）       （下行）
存货↑    被动补库存    主动补库存
         （上行）       （高位）

具体标准：
- 主动去库存：存货同比<-5%，营收同比<0% → 景气底部
- 被动补库存：存货同比>5%，营收同比>10% → 景气上行 ✓
- 主动补库存：存货同比>10%，营收同比>5% → 景气高位（警惕）
- 被动去库存：存货同比<0%，营收同比<-5% → 景气下行 ✗
```

---

### 7.2 库存周转天数

**指标定义**：存货周转天数 = 存货 / (营业成本/365)

**数据来源**：
| 平台 | 获取路径 |
|-----|---------|
| **Wind** | 行业 → 行业对比 → 营运能力 → 存货周转天数 |
| **同花顺** | 行业数据 → 营运能力 → 存货周转天数 |

**判断标准**：
```
库存积压判断：
- 库存周转天数环比下降 → 去库顺畅 ✓
- 库存周转天数环比上升 < 10% → 正常波动
- 库存周转天数环比上升 > 20% → 库存积压 ✗
- 库存周转天数环比上升 > 30% → 严重积压（红线）
```

**行业基准值**（仅供参考）：
| 行业 | 合理天数 |
|-----|---------|
| 白酒 | 200-400天（需陈化） |
| 家电 | 30-60天 |
| 电子 | 40-70天 |
| 钢铁 | 20-40天 |
| 服装 | 90-150天 |

## 八、定性评估指标

### 8.1 政策环境

**判断方法**：主观评估 + 政策追踪

**信息来源**：
| 来源 | 获取路径 | 更新频率 |
|-----|---------|---------|
| **中央政策** | 中国政府网 www.gov.cn | 实时 |
| **部委政策** | 工信部、发改委、科技部等官网 | 实时 |
| **政治局会议** | 新华社、人民日报 | 季度 |
| **行业规划** | 各部委官网"政策法规"栏目 | 不定期 |

**评分标准（5分制）**：
| 分数 | 政策特征 | 示例 |
|-----|---------|------|
| **5分** | 国家战略、政策强力支持、有补贴 | 新能源、半导体、创新药 |
| **3分** | 政策中性、无明确限制或扶持 | 大部分制造业 |
| **1分** | 政策限制、整顿、监管趋严 | 教育培训、地产、游戏 |

**重点监控**：
- 五年规划（十四五/十五五）中的重点产业
- 政治局会议/中央经济工作会议定调
- 行业专项政策（如"双碳"、"国产替代"）

---

### 8.2 商业模式

**判断方法**：主观评估

**评估维度**：
| 维度 | 优秀特征 | 较差特征 |
|-----|---------|---------|
| **盈利模式** | 可持续、可复制、规模效应 | 依赖补贴、一次性收入 |
| **客户粘性** | 高频复购、转换成本高 | 一次性交易、价格敏感 |
| **现金流** | 预收款模式、账期短 | 垫资、账期长 |
| **网络效应** | 用户越多价值越大 | 无网络效应 |

**评分标准（5分制）**：
```
5分（优秀）：SaaS订阅、品牌消费品、平台型
  - 例：茅台（品牌溢价）、腾讯（网络效应）

3分（良好）：制造业、To B服务
  - 例：家电龙头、工业品供应商

1分（较差）：工程承包、贸易流通
  - 例：建筑施工、钢贸
```

**信息来源**：
- 上市公司年报"业务模式"章节
- 券商深度研报
- 行业专家访谈

---

### 8.3 进入壁垒

**判断方法**：多维度评估

**壁垒类型及判断**：
| 壁垒类型 | 判断指标 | 数据来源 |
|---------|---------|---------|
| **技术壁垒** | 研发投入占比、专利数量 | Wind财务数据、专利局 |
| **品牌壁垒** | 品牌价值排名、广告费用率 | 世界品牌实验室、财报 |
| **渠道壁垒** | 销售网点数量、经销商数量 | 年报、调研 |
| **规模壁垒** | 单位成本、市占率 | Wind、行业协会 |
| **政策壁垒** | 牌照数量、准入门槛 | 监管部门官网 |

**具体查询**：
- **研发投入占比**：Wind → 财务指标 → 研发费用/营业收入
- **专利数量**：国家知识产权局 → 专利检索 → 按公司名检索
- **品牌价值**：世界品牌实验室 www.worldbrandlab.com

**评分标准（5分制）**：
```
5分：3项以上壁垒 + 极难复制
  - 例：茅台（品牌+地理+工艺）、片仔癀（配方+资源）

3分：1-2项壁垒 + 有一定门槛
  - 例：家电龙头（规模+渠道）

1分：无明显壁垒 + 容易进入
  - 例：低端制造、贸易
```

---

### 8.4 护城河深度

**判断方法**：综合主观评估

**评估框架（巴菲特/晨星体系）**：
| 护城河类型 | 核心问题 | 判断依据 |
|-----------|---------|---------|
| **无形资产** | 品牌溢价能力如何？ | 提价后销量变化、品牌调研 |
| **转换成本** | 客户换品牌成本有多高？ | 客户留存率、行业惯例 |
| **网络效应** | 用户越多越好用吗？ | 用户数增长、平台活跃度 |
| **成本优势** | 成本比竞争对手低多少？ | 单位成本对比、规模经济 |

**评分标准（5分制）**：
```
5分（深厚）：拥有2种以上护城河，10年内难以攻破
  - 例：茅台、腾讯、片仔癀

3分（一般）：拥有1种护城河，5年内可维持
  - 例：海天味业、格力电器

1分（薄弱）：无明显护城河，随时可能被颠覆
  - 例：普通制造业、同质化竞争行业
```

**信息来源**：
- 公司年报、券商研报
- 竞争对手对比分析
- 行业专家访谈

## 九、红线触发指标

### 9.1 毛利率连续下滑

**监控指标**：行业平均毛利率季度环比/同比

**数据来源**：
```
Wind → 行业对比 → 毛利率（季度）→ 导出近8个季度数据
```

**红线标准**：
```
立即调出：
- 毛利率连续4季度下滑，且累计下降 > 5个百分点

预警观察：
- 毛利率连续2季度下滑，且累计下降 > 3个百分点
```

**监控频率**：季报发布后1周内

---

### 9.2 行业收入连续下滑

**监控指标**：行业营业收入同比增速

**数据来源**：
```
Wind → 行业对比 → 营业收入同比（季度）
或
国家统计局 → 工业增加值增速（月度）
```

**红线标准**：
```
立即调出：
- 行业收入连续2季度同比下滑 > 20%

预警观察：
- 行业收入连续2季度同比下滑 > 10%
```

**监控频率**：季报/月度数据发布后

---

### 9.3 估值历史高位

**监控指标**：PE/PB历史分位

**数据来源**：
```
理杏仁 → 申万行业 → PE/PB分位（5年）
```

**红线标准**：
```
立即调出：
- PE或PB处于历史95%分位以上，且维持1个月

预警观察：
- PE或PB处于历史80%分位以上
```

**监控频率**：月度（每月第一个交易日）

---

### 9.4 资金持续流出

**监控指标**：北向资金/主力资金净流入

**数据来源**：
| 数据类型 | 获取渠道 |
|---------|---------|
| **北向资金** | 东方财富 → 数据中心 → 北向资金 → 行业分布 |
| **主力资金** | 同花顺 → 行业资金流向 |
| **Wind** | 资金流向 → 行业资金流 |

**具体查询**：
```
东方财富网：data.eastmoney.com → 北向资金 → 行业资金流向
同花顺：行情 → 行业板块 → 资金流向
```

**红线标准**：
```
立即调出：
- 北向资金+主力资金连续8周净流出，且累计流出 > 100亿

预警观察：
- 北向资金+主力资金连续4周净流出
```

**监控频率**：每周五收盘后
