# 机会资产投资策略

**定位**：占总仓位30%，捕捉优质股错杀机会
**目标**：通过1-2年持有期，获得单次40-60%收益（年化20-25%）
**核心理念**：在市场恐慌时买入临时困境的优质公司，等待估值修复
