# 七、机会资产看板

Excel/表格模板，用于跟踪机会资产的关键指标。

## 关键字段

- **类型**：情绪错杀/周期错杀/事件错杀/政策错杀
- **PE分位**：当前PE在历史的百分位（回到50分位以上考虑退出）
- **剩余空间**：距离目标价的上涨空间
- **催化剂**：买入时识别的修复因素
- **操作计划**：继续持有/准备退出/立即止损
