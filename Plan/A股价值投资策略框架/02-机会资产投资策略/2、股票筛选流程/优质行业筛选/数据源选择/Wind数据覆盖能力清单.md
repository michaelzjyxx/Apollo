## 二、详细指标清单

### ✅ Wind可以获取所有指标（26个）

#### 1. 竞争格局指标（4/4）✓✓✓

**✓ 1.1 行业集中度（CR5）**
- **获取路径**：Wind → 行业 → 行业分析 → 市场集中度 → CR5
- **数据特点**：
  - CR3/CR5/CR10/HHI指数齐全
  - 可按营收/市值/产量多维度计算
  - 历史数据完整（20年+）
  - 自动更新（季度）
  - 可自定义计算口径
- **独家优势**：
  - 覆盖申万全部134个二级行业
  - 可查看CR5的历史变化趋势
  - 支持自定义企业范围（如仅上市公司/全行业）

**✓ 1.2 龙头市场份额变化**
- **获取路径**：Wind → 个股 → 公司研究 → 市场地位 → 市场份额
- **数据特点**：
  - 龙头企业市场份额（营收/销量）
  - 历史趋势图（可回溯15年+）
  - 可对比行业前10名份额变化
  - 自动计算份额增减
- **独家优势**：
  - 数据来源多样：公司公告+行业协会+Wind估算
  - 部分行业有销量市场份额（如汽车、家电）
  - 可查看细分市场份额（如高端白酒、入门白酒）

**✓ 1.3 价格体系稳定性**
- **获取路径**：Wind → 商品 → 商品价格 → 工业品/消费品价格
- **数据特点**：
  - 覆盖5000+商品价格
  - 工业品：钢材、水泥、化工、有色等
  - 消费品：接入统计局CPI分项
  - 日度/周度/月度更新
  - 可计算价格波动率（CV）
- **独家优势**：
  - 最全的商品价格数据库（远超iFinD）
  - 整合期货+现货价格
  - 可直接计算价差、基差
  - 这是Wind相比其他平台的核心优势之一！

**✓ 1.4 产能利用率**
- **获取路径**：Wind → 行业 → 行业数据 → 产能利用率
- **数据特点**：
  - 接入统计局官方数据
  - 覆盖30+重点行业
  - 季度数据（部分月度）
  - 历史数据完整
- **独家优势**：
  - 部分行业有Wind自建产能利用率模型
  - 可查看细分行业（如螺纹钢、水泥分区域）
  - 覆盖行业最全

#### 8. 定性评估指标（4/4）✓✓✓

**✓ 8.1 政策环境（辅助数据最全）**
- **获取路径**：Wind → 资讯 → 政策法规 → 行业政策
- **数据特点**：
  - 全面整合政策新闻
  - 按行业分类推送
  - 政策影响评级（利好/中性/利空）
  - 政策文件原文
- **独家优势**：
  - Wind资讯库最全（覆盖政府网站+媒体）
  - 可设置行业政策预警
  - 可查看历史政策对行业影响（事件研究）
  - 可查看券商政策解读报告
- **注意**：最终仍需主观判断，但Wind提供最全辅助信息

**✓ 8.2 商业模式（辅助数据最全）**
- **获取路径**：Wind → 公司研究 → 业务模式
- **数据特点**：
  - 整合公司年报业务模式章节
  - 可查看商业模式关键指标：
    - 客户集中度（前5大客户占比）
    - 预收款/营收（预收款模式）
    - 应收账款周转天数（账期）
    - 研发费用率（技术驱动）
- **独家优势**：
  - 可量化评估商业模式质量（Wind商业模式评分）
  - 可对比同行业商业模式差异
  - 可查看商业模式变化（如从to C转to B）
- **注意**：仍需主观判断，但Wind提供最多量化支撑

**✓ 8.3 进入壁垒（辅助数据最全）**
- **获取路径**：Wind → 行业分析 → 竞争壁垒
- **可获取的量化数据**：
  - **技术壁垒**：研发费用率、专利数量、研发人员占比
  - **品牌壁垒**：品牌价值排名（接入多个榜单）、广告费用率
  - **渠道壁垒**：销售网点数量（部分行业）、经销商数量
  - **规模壁垒**：单位成本、产能规模、市占率
  - **政策壁垒**：牌照数量、准入门槛（行业研报）
- **独家优势**：
  - Wind整合多维度壁垒数据
  - 可对比壁垒强度（行业间、公司间）
  - 可查看壁垒变化趋势（如研发投入增速）
  - **数据维度最全，远超其他平台**

**✓ 8.4 护城河深度（辅助数据最全）**
- **获取路径**：Wind → 公司研究 → 护城河分析
- **可获取的量化数据**：
  - **无形资产**：品牌价值、专利数量、毛利率（品牌溢价）
  - **转换成本**：客户留存率、NPS（部分）、合同期限
  - **网络效应**：用户数增长、平台活跃度（互联网）
  - **成本优势**：单位成本对比、规模效应、产能利用率
- **独家优势**：
  - Wind有护城河评分模型（参考晨星体系）
  - 可查看护城河宽度vs ROE相关性
  - 可分析护城河持续性（历史ROE稳定性）
  - **这是Wind相比其他平台的重要优势**
