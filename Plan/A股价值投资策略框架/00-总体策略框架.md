# A股价值投资策略框架

**策略定位**：双轨制价值投资策略
- 核心资产（70%仓位）：长期持有5-10年
- 机会资产（30%仓位）：错杀修复套利，持仓1-2年

## 投资目标与资金方案

### 终极目标
**第11年起，每年获得100万投资收益**

### 推荐方案（混合策略）
```
初始投入：25万
年度追加：12万（月均1万元）
目标年化：15%（核心资产12% + 机会资产22%）
投资期限：12-15年

资产配置：
- 核心资产：70%（长期持有优质股）
- 机会资产：30%（错杀修复套利）

预期结果：
- 第12年末资产：约450万
- 第13年年化收益：67.5万/年
- 第15年末资产：约700万
- 第16年年化收益：105万/年 ✅
```

### 风险情景分析

**情景1：收益不达预期（年化12%）**
- 第12年资产：约380万，第13年年收益45.6万
- 应对：延长至第18年，资产达710万，年收益85万 ✅

**情景2：遇到熊市（第3-5年持续下跌-40%）**
- 第12年资产：约310万
- 应对：坚持定投不止损，第15-18年牛市修复，第18年资产可达550万 ✅

**情景3：超预期（年化18%）**
- 第12年资产：约620万，年收益112万
- 策略：提前达成目标，第10年开始降低风险仓位

**情景4：极端情况（系统性风险）**
- 核心资产遭遇黑天鹅（单只腰斩）
- 影响：若15%仓位腰斩，总资产损失7.5%
- 应对：严格遵守15%单只上限，触发复审机制及时止损

**情景5：个人现金流中断**
- 失业/重大支出导致无法定投2-3年
- 影响：第12年资产降至约380万（少追加36万）
- 应对：延长投资期限2-3年，保持已投资金持续复利

## 资本配置核心原则

### 1. 核心驱动力
首要目标：寻找并持有价格显著低于内在价值的优质公司，获取价值回归与成长收益。
次要考量：将市场整体估值水平作为评估"机会丰度"的参考背景，而非仓位决策的唯一开关。

### 2. 资金配置节奏
高估值（机会稀缺）市场：提高筛选标准，放缓建仓速度，强调安全边际。
低估值（机会充裕）市场：更积极地执行投资计划，对符合条件的标的敢于下注。
总仓位：由可识别的高价值机会数量及其确定性自然形成，不基于市场方向预测进行主动择时。

## 双轨资产配置

### 核心资产（70%）
**目标**：长期持有优质公司，享受复利增长
- 持仓数量：5-8只
- 单只仓位：超级核心（茅台、招行）10-15%，普通核心8-12%
- 预期年化收益：12%
- 持有周期：5-10年

**分散原则**：
- 行业分散：覆盖3-5个不相关行业，单一行业≤30%
- 个股分散：5-8只，单一个股≤15%
- 目的：防范个体公司黑天鹅与行业性风险

**集中原则**（有条件）：
- 集中条件：仅在"行业需求永续、护城河极深、管理层可信赖、估值处历史极低分位"四项条件同时满足时
- 集中上限：超级核心资产持仓上限可提升至15%（如茅台、招行）
- 目的：在严格风控纪律下，提升顶级机会的资本回报效率

详见：[核心资产投资策略](./02-核心资产投资策略.md)

### 机会资产（30%）
**目标**：捕捉优质股错杀机会，赚取估值修复收益
- 持仓数量：3-5只
- 单只仓位：5-8%
- 预期收益：单次40-60%（年化20-25%）
- 持有周期：1-2年

**分散原则**：
- 同时持有3-5只错杀标的
- 单一个股≤8%
- 避免同一行业多个机会标的（降低行业系统性风险）

**不集中原则**：
- 机会资产单只上限严格控制在8%，不集中持仓（因持有周期短，不确定性高）

详见：[机会资产投资策略](./03-机会资产投资策略.md)

## 风险控制体系

### 1. 首要目标：规避本金永久性损失
- 深度研究是首要风控：每一笔投资必须能清晰陈述其"价值基石"与"关键假设"。
- 安全边际是核心保障：支付价格必须提供足够的犯错缓冲。

### 2. 过程风控
- 事前：严格执行筛选流程与仓位上限规则。
- 事中：持续监控投资的"关键假设"是否成立，而非仅关注股价波动。
- 事后：交易复盘聚焦于"决策过程质量"与"认知偏差纠正"。

## 执行纪律

### 1. 每月操作清单

**第1周：核心资产维护**
```
周一:
□ 定投1万元 → 按计划买入核心资产
□ 检查核心资产池估值（PE/PB百分位）

周末:
□ 阅读1-2份行业研报
□ 更新核心资产看板
```

**第2周：机会资产扫描**
```
周一:
□ 扫描机会资产候选池（20-30只）
□ 标记PE<20分位的标的

周三:
□ 深度研究2-3只"接近买点"的标的
□ 分析催化剂、计算目标价

周末:
□ 决策：是否有值得买入的机会
```

**第3周：持仓检查**
```
周一:
□ 检查机会资产是否达到退出条件
□ 计算各标的预期收益 vs 新机会

周末:
□ 执行：卖出 or 继续持有
□ 释放资金等待新机会
```

**第4周：月度复盘**
```
周末:
□ 计算当月收益率
□ 更新资产配置比例（核心70% vs 机会30%）
□ 检查是否需要再平衡
□ 记录投资日志（买卖决策、心得）
```

### 2. 季度操作清单

**财报季（4月/7月/10月/次年1月）**
```
财报发布后3天内:
□ 核心资产逐一检查财报
□ 重点关注：ROE、营收增速、现金流
□ 标记：超预期 / 符合预期 / 低于预期

财报季后1周:
□ 决策：是否调仓
□ 执行：卖出基本面恶化的
□ 执行：加仓超预期的

机会资产:
□ 检查持有天数（>180天开始关注退出）
□ 更新目标价（基于新财报）
```

### 4. 纪律要求

**严格执行框架**：
- 遵守仓位上限规则（核心≤15%，机会≤8%）
- 遵守复审纪律（定期+触发式）
- 遵守止盈止损规则

**完整记录**：
- 每笔投资的买入逻辑、关键假设
- 核心资产：预期持有期、护城河分析
- 机会资产：目标价、催化剂、退出计划

**定期复盘**：
- 每季度复盘投资决策质量
- 每年复盘框架有效性并迭代优化
- 记录认知偏差和改进措施

### 5. 双轨策略关键差异总结

| 维度 | 核心资产（70%） | 机会资产（30%） |
|------|----------------|----------------|
| **选股标准** | ROE>15%，护城河强 | ROE>12%，临时困境 |
| **买入时机** | PE<30分位+PEG<1 | 必须等到恐慌低估 |
| **买入方式** | 分批建仓3-6个月 | 快速建仓1-3个月 |
| **单只仓位** | 8-15% | 5-8% |
| **持有周期** | 5-10年 | 1-2年 |
| **监控频率** | 季度 | 周度 |
| **退出条件** | 基本面恶化 | 涨幅达标/时间到期 |
| **收益预期** | 年化12% | 单次40-60% |
| **风险等级** | 低 | 中 |
| **精力投入** | 低（季度看一次） | 高（每周跟踪） |
