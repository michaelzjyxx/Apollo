## 二、详细指标清单

### ✅ iFinD可以获取的指标（23个）

#### 1. 竞争格局指标（3/4）

**✓ 1.1 行业集中度（CR5）**
- **获取路径**：iFinD → 数据浏览器 → 行业数据 → 集中度指标
- **数据特点**：
  - 可自定义计算CR3/CR5/CR10
  - 按营收或市值计算
  - 可导出历史数据
  - 更新：季度更新
- **优势**：比理杏仁强，支持自定义计算

**✓ 1.2 龙头市场份额变化**
- **获取路径**：iFinD → 个股F10 → 行业地位 → 市场份额
- **数据特点**：
  - 提供龙头企业市场份额
  - 可查看历史变化趋势
  - 部分行业数据较全
- **注意**：依赖上市公司披露，非所有行业都有

**❌ 1.3 价格体系稳定性**
- **iFinD无专门的行业价格数据库**
- **替代方案**：
  - iFinD有部分大宗商品期货价格
  - 但工业品现货价格需依赖卓创、生意社
  - CPI/PPI有（统计局数据接入）

**✓ 1.4 产能利用率**
- **获取路径**：iFinD → 宏观数据 → 工业数据 → 产能利用率
- **数据特点**：
  - 接入统计局数据
  - 部分重点行业有数据（钢铁、水泥等）
  - 月度/季度更新
- **注意**：覆盖行业不如Wind全面

---

#### 2. 盈利能力指标（2/2）✓✓

**✓ 2.1 行业平均ROE**
- **获取路径**：iFinD → 数据浏览器 → 行业财务指标 → ROE
- **数据特点**：
  - 可选市值加权/等权平均
  - TTM、LYR可选
  - 历史数据完整（可回溯10年+）
  - 可导出Excel
- **优势**：比理杏仁数据完整，可自定义加权方式

**✓ 2.2 行业平均毛利率**
- **获取路径**：iFinD → 行业数据 → 盈利能力 → 毛利率
- **数据特点**：
  - 营收加权平均
  - 可查看季度/年度数据
  - 可绘制趋势图
  - 导出功能强大

---

#### 3. 成长性指标（2/2）✓✓

**✓ 3.1 行业收入增速**
- **获取路径**：iFinD → 行业数据 → 成长能力 → 营收同比增速
- **数据特点**：
  - TTM同比、单季同比
  - 可按申万一级/二级分类
  - 历史数据齐全
  - 可对比多个行业
- **补充**：也可获取统计局工业增加值数据

**✓ 3.2 行业利润增速**
- **获取路径**：iFinD → 行业数据 → 成长能力 → 净利润同比增速
- **数据特点**：
  - TTM、单季可选
  - 可计算复合增长率（CAGR）
  - 支持自定义时间区间

---

#### 4. 现金流指标（2/2）✓✓

**✓ 4.1 经营性现金流/净利润**
- **获取路径**：iFinD → 行业数据 → 现金流指标 → 经营现金流/净利润
- **数据特点**：
  - 直接提供比值
  - 行业整体加权数据
  - 可查看个股明细
  - 季度更新
- **优势**：比理杏仁方便，不需手工计算

**✓ 4.2 资本支出强度**
- **获取路径**：iFinD → 行业数据 → 现金流量表 → Capex、折旧
- **数据特点**：
  - 可提取Capex和折旧数据
  - 需手工计算比值（或用公式工具）
  - 可导出批量计算
- **注意**：没有直接的"资本支出强度"指标，需自己算

---

#### 5. 估值指标（2/2）✓✓

**✓ 5.1 行业PE/PB历史分位**
- **获取路径**：iFinD → 行业估值 → 申万行业 → PE/PB Band
- **数据特点**：
  - PE-TTM、PB历史分位
  - 可选3年/5年/10年/全部
  - 分位数图表（类似理杏仁）
  - 实时更新
- **对比**：
  - 功能与理杏仁相当
  - 但iFinD数据历史更长
  - 可导出数据做自定义分析

**✓ 5.2 PEG（市盈率相对盈利增速比）**
- **获取路径1**：iFinD → 行业估值 → PEG（部分行业直接提供）
- **获取路径2**：
  - PE：行业估值 → PE-TTM
  - 增速：一致预期 → 净利润增速（未来1年）
  - 手工计算：PEG = PE / 增速
- **数据特点**：
  - 部分行业有一致预期增速
  - 成长行业覆盖较好（科技、医药）
  - 周期行业一致预期较少

---

#### 6. 景气度领先指标（4/4）✓✓

**✓ 6.1 制造业PMI**
- **获取路径**：iFinD → 宏观数据 → 景气指数 → 制造业PMI
- **数据特点**：
  - 接入统计局+财新PMI
  - 历史数据完整
  - 分项指数齐全（新订单、生产、库存等）
  - 可设置预警
- **优势**：比统计局官网方便，数据在一个平台

**✓ 6.2 新订单指数**
- **获取路径**：iFinD → 宏观数据 → PMI → 新订单指数
- **数据特点**：
  - 统计局+财新双口径
  - 可查看分行业新订单（部分）
  - 历史走势图

**✓ 6.3 社融/M2增速**
- **获取路径**：iFinD → 宏观数据 → 货币金融 → M2/社融
- **数据特点**：
  - 接入央行数据
  - M2、M1、社融存量、增速齐全
  - 可查看历史走势
  - 每月自动更新
- **优势**：比央行官网方便，可直接绘图分析

**✓ 6.4 PPI/CPI同比**
- **获取路径**：iFinD → 宏观数据 → 物价 → PPI/CPI
- **数据特点**：
  - 接入统计局数据
  - CPI/PPI分项齐全（食品、黑色金属等）
  - 可导出分析
  - 自动更新
- **优势**：比统计局官网数据更易用

---

#### 7. 周期位置判断指标（2/2）✓✓

**✓ 7.1 存货同比**
- **获取路径**：iFinD → 行业数据 → 资产负债表 → 存货 → 同比增速
- **数据特点**：
  - 行业整体存货同比
  - 可查看季度数据
  - 可与营收同比对比（库存周期分析）
- **优势**：直接提供行业数据，不需手工汇总

**✓ 7.2 库存周转天数**
- **获取路径**：iFinD → 行业数据 → 营运能力 → 存货周转天数
- **数据特点**：
  - 行业加权平均周转天数
  - 可查看历史趋势
  - 环比、同比变化
- **应用**：可直接判断库存积压情况

---

#### 8. 定性评估指标（2/4，部分支持）

**✓ 8.1 进入壁垒（部分数据支持）**
- **可获取的辅助数据**：
  - 研发投入占比：iFinD → 财务指标 → 研发费用/营收
  - 专利数量：需外部查询（国家知识产权局）
  - 品牌价值：部分接入第三方榜单
- **主观判断**：仍需结合年报和研报

**✓ 8.2 护城河深度（部分数据支持）**
- **可获取的辅助数据**：
  - 毛利率对比（成本优势）
  - 客户集中度（转换成本）
  - 市场份额（规模优势）
- **主观判断**：需综合评估

**❌ 8.3 政策环境**
- **iFinD无专门的政策评估工具**
- **替代**：有政策新闻推送，需人工判断

**❌ 8.4 商业模式**
- **需主观判断**
- **辅助**：iFinD有行业研报库，可参考

---

#### 9. 红线触发指标（4/4）✓✓

**✓ 9.1 毛利率连续下滑**
- **监控路径**：iFinD → 行业数据 → 毛利率（季度）→ 导出监控
- **数据特点**：
  - 完整季度历史数据
  - 可设置条件预警（高级功能）
  - 支持批量监控
- **优势**：可设置自动预警（需配置）

**✓ 9.2 行业收入连续下滑**
- **监控路径**：iFinD → 行业数据 → 营收同比（季度）
- **数据特点**：
  - 季度数据齐全
  - 可设置条件筛选（如连续2季度下滑>20%）
  - 支持导出

**✓ 9.3 估值历史高位**
- **监控路径**：iFinD → 行业估值 → PE/PB分位
- **数据特点**：
  - 实时查看分位数
  - 可设置估值预警（如分位>95%自动提醒）
  - 支持自定义分位区间
- **优势**：可配置自动预警，比理杏仁智能

**✓ 9.4 资金持续流出**
- **监控路径**：iFinD → 资金流向 → 行业资金流
- **数据特点**：
  - 北向资金分行业流向
  - 主力资金分行业数据
  - 可查看周度、月度累计
  - 实时更新
- **优势**：比东方财富更专业，可导出分析

### ❌ iFinD无法直接获取的指标（3个）

**❌ 1. 价格体系稳定性（工业品现货价格）**

- **替代方案**：
  - iFinD有期货价格（部分参考）
  - 需补充卓创资讯、生意社等专业平台

**❌ 2. 政策环境（主观判断）**
- **原因**：政策评估需要主观判断
- **iFinD辅助**：
  - 有政策新闻推送
  - 有行业研报库
  - 但最终需人工判断

**❌ 3. 商业模式（主观判断）**
- **原因**：商业模式评估需深度研究
- **iFinD辅助**：
  - 可查看公司年报
  - 可看研报分析
  - 但需人工评估

## 五、iFinD使用建议

### 5.1 核心应用场景

**最适合的场景**：
1. **全流程行业筛选** ⭐⭐⭐⭐⭐
   - 从宏观→行业→个股，一站式
   - 无需多平台跳转
   - 数据覆盖88%

2. **宏观+行业联动分析** ⭐⭐⭐⭐⭐
   - PMI+行业景气度
   - 社融+地产/金融行业
   - 一个平台搞定

3. **量化筛选与回测** ⭐⭐⭐⭐
   - 批量导出数据
   - 自定义指标计算
   - 历史回测

4. **红线监控与预警** ⭐⭐⭐⭐
   - 可设置自动预警
   - 邮件/短信提醒
   - 实时监控

---

### 5.2 典型使用流程

**行业筛选完整流程**：

```
Step 1: 宏观判断
iFinD → 宏观数据 → PMI、社融、PPI
→ 判断经济周期位置

Step 2: 行业景气度
iFinD → 行业数据 → 营收增速、利润增速、ROE
→ 筛选出高景气行业

Step 3: 竞争格局
iFinD → 行业数据 → CR5、毛利率趋势
→ 剔除竞争恶化行业

Step 4: 估值筛选
iFinD → 行业估值 → PE/PB分位
→ 筛选出低估行业

Step 5: 资金验证
iFinD → 资金流向 → 北向资金行业分布
→ 确认资金流入

Step 6: 导出分析
iFinD → 导出Excel → 综合评分
→ 确定目标行业
```

**全流程在一个平台完成！**

---

### 5.3 高频使用技巧

**技巧1：快速查看行业景气度**
```
iFinD → 数据浏览器 → 行业对比
→ 选择"申万行业"
→ 添加字段：营收增速、ROE、PE分位
→ 一键生成对比表
```

**技巧2：设置估值预警**
```
iFinD → 行业估值 → PE分位
→ 右键 → 设置预警
→ 条件：PE分位 > 95%
→ 通知方式：邮件/短信
```

**技巧3：批量监控红线指标**
```
iFinD → 数据浏览器 → 自定义筛选
→ 条件1：毛利率环比下降 > 5%
→ 条件2：营收同比 < -20%
→ 条件3：PE分位 > 95%
→ 保存为"红线监控"方案
→ 每季度运行一次
```

**技巧4：宏观行业联动分析**
```
iFinD → 宏观数据 → 社融同比
→ 叠加 → 行业数据 → 地产行业营收增速
→ 查看领先/滞后关系
→ 预判行业拐点
```
