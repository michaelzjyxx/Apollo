# 第一层：行业筛选（与核心资产共用）

## 一、筛选框架总览

行业筛选遵循"自上而下"的分析逻辑，通过多维度评估体系，筛选出具备长期投资价值的优质行业。

```
行业池 → 政策环境筛选 → 商业模式筛选 → 竞争格局筛选 → 景气度筛选 → 目标行业池
```

---

## 二、剔除标准（负面清单）

### 2.1 政策风险维度
**剔除行业类型**：
- **政策不确定性高的行业**
  - 政策频繁调整的行业（如房地产调控、教育培训等）
  - 政策导向不明确的新兴行业
  - 受政府定价管制严重的行业

- **政策转向风险大的行业**
  - 逆城市化、逆全球化趋势的行业
  - 碳中和、环保政策下的高污染、高耗能行业
  - 产业政策明确限制发展的行业

### 2.2 商业模式维度
**剔除行业类型**：
- **商业模式不清晰的行业**
  - 盈利模式不可持续的行业
  - 严重依赖补贴或政策红利的行业
  - 缺乏规模效应和网络效应的行业

- **竞争过度激烈的行业**
  - 进入壁垒低、同质化竞争严重的行业
  - 价格战频发、毛利率持续下滑的行业
  - 市场份额高度分散（CR5 < 20%）的行业

### 2.3 供需格局维度
**剔除行业类型**：
- **产能严重过剩的行业**
  - 产能利用率长期低于70%的行业
  - 供给增速显著高于需求增速的行业
  - ROE持续低于社会平均水平的行业

- **需求长期萎缩的行业**
  - 受技术替代威胁严重的传统行业
  - 人口结构变化导致需求下滑的行业
  - 消费升级下被淘汰的低端行业

## 三、保留标准（正面清单）

### 3.1 行业格局稳定性
**优选行业特征**：
- **竞争格局清晰**
  - 行业集中度高（CR5 > 40%）或稳步提升
  - 龙头企业市场份额稳定或扩大
  - 价格体系稳定，避免恶性竞争

- **进入壁垒明显**
  - 技术壁垒：核心技术专利、研发投入高
  - 品牌壁垒：消费者认知度高、品牌忠诚度强
  - 渠道壁垒：销售网络完善、客户粘性高
  - 规模壁垒：规模经济效应显著
  - 政策壁垒：牌照、资质等准入门槛

### 3.2 长期需求确定性
**优选行业特征**：
- **刚性需求驱动**
  - 必需消费品行业（食品饮料、医药等）
  - 基础设施行业（水电气、通信等）
  - 老龄化带来的需求增长（医疗、养老等）

- **增量需求明确**
  - 城镇化、消费升级带来的需求扩容
  - 技术进步带来的新兴需求
  - 国产替代、自主可控带来的市场空间

### 3.3 产业升级与消费升级
**优选行业方向**：
- **产业升级受益行业**
  - 高端制造：半导体、新能源、高端装备
  - 数字经济：云计算、人工智能、物联网
  - 绿色经济：新能源、环保、节能技术

- **消费升级受益行业**
  - 品质消费：高端白酒、化妆品、品牌服装
  - 服务消费：医疗美容、职业教育、休闲娱乐
  - 健康消费：创新药、医疗器械、健身运动

### 3.4 行业护城河评估
**护城河类型识别**：
- **成本优势护城河**
  - 规模效应带来的成本优势
  - 独特资源（矿产、地理位置等）
  - 工艺流程优化带来的效率提升

- **网络效应护城河**
  - 平台型企业的双边市场效应
  - 用户规模带来的价值增值
  - 生态系统的协同效应

- **无形资产护城河**
  - 强品牌认知与溢价能力
  - 核心专利与技术壁垒
  - 监管许可与行业资质

- **转换成本护城河**
  - 客户迁移成本高
  - 系统集成度高、替换难度大
  - 长期服务关系与信任积累

## 四、行业景气度评估

### 4.1 量化评估指标（相对比较法）
**核心指标体系**：

| 维度 | 核心指标 | 优选标准（相对比较） |
|------|---------|---------|
| **成长性** | 行业收入增速 | 高于全A中位数，或处于自身历史70%分位以上 |
| | 行业利润增速 | 高于收入增速（利润弹性>1） |
| **盈利能力** | 行业平均毛利率 | 高于自身5年均值，且环比改善 |
| | 行业平均ROE | 高于全A中位数（约8%），或自身历史50%分位以上 |
| **现金流** | 经营性现金流/净利润 | >80%，且连续3年为正 |
| | 资本支出/折旧 | <2（避免过度投资） |
| **估值水平** | 行业PE/PB历史分位 | <50%分位（5年） |
| | PEG | <1.5，或低于同类行业 |

> **说明**：不同行业特性差异大，不宜用统一绝对值。制造业ROE>12%即优秀，消费行业>18%才达标。建议采用「自身历史比较+同类横向比较」双维度。

### 4.2 景气度领先指标体系

**宏观领先指标**：
| 指标 | 数据来源 | 领先周期 | 适用行业 |
|-----|---------|---------|--------|
| 制造业PMI | 国家统计局 | 1-2个月 | 制造业全行业 |
| 新订单指数 | 国家统计局 | 2-3个月 | 工业品、设备 |
| 社融/M2增速 | 央行 | 3-6个月 | 金融、地产 |
| PPI同比 | 国家统计局 | 同步 | 周期品 |
| CPI同比 | 国家统计局 | 同步 | 消费品 |

**行业领先指标**（按行业类型）：
| 行业类型 | 领先指标 | 数据来源 |
|---------|---------|--------|
| 消费品 | 社零增速、CPI分项、渠道库存 | 统计局、行业协会 |
| 周期品 | 产品价格、开工率、库存天数 | Wind、卓创资讯 |
| 科技成长 | 订单增速、产能利用率、研发投入 | 财报、产业链调研 |
| 金融地产 | 社融、利率、商品房销售 | 央行、统计局 |

### 4.3 周期位置量化判断

**库存周期定位**（基于存货同比+营收同比）：
```
              营收↑        营收↓
存货↓    主动去库存    被动去库存
         （景气底部）   （景气下行）
存货↑    被动补库存    主动补库存
         （景气上行）   （景气高位）
```

**具体判断标准**：
- **景气上行期**：PMI>50且连续3月上行、新订单指数>51、价格环比上涨
- **景气高位期**：产能利用率>85%、固定资产投资加速、价格同比涨幅收窄
- **景气下行期**：PMI<50或连续3月下行、库存天数上升>20%、价格环比下跌
- **景气底部期**：产能利用率<70%、企业数量减少、价格企稳或反弹

**政策周期判断**：
- 产业政策扶持周期（新能源、半导体等）
- 监管政策收紧周期（互联网平台、教育培训等）
- 财政政策刺激周期（基建、地产等）

## 五、数据获取与工具

### 5.1 核心数据平台

| 平台 | 类型 | 主要用途 | 成本 |
|-----|-----|---------|------|
| **Wind** | 专业终端 | 行业财务数据、估值、比较 | 付费（约2-3万/年） |
| **同花顺iFinD** | 专业终端 | 行业数据、个股筛选 | 付费（约1-2万/年） |
| **东方财富Choice** | 专业终端 | 行业数据、资金流向 | 付费（约5000/年） |
| **理杏仁** | 网站 | 行业估值分位、ROE等 | 免费/付费 |
| **国家统计局** | 官方网站 | PMI、工业数据、社零 | 免费 |
| **中国人民银行** | 官方网站 | 社融、M2、利率 | 免费 |

### 5.2 数据查询路径

**行业财务数据**（以Wind为例）：
```
Wind → 行业 → 行业对比 → 选择申万行业 → 财务指标
```

**行业估值分位**（以理杏仁为例）：
```
理杏仁 → 指数估值 → 申万行业指数 → PE/PB历史分位
```

**景气度数据**：
```
国家统计局官网 → 月度数据 → 工业 → 制造业PMI
中国人民银行官网 → 统计数据 → 社会融资规模
```

**行业高频数据**：
| 数据类型 | 查询渠道 |
|---------|--------|
| 产品价格 | 卓创资讯、生意社、百川盈孚 |
| 开工率 | 隆众资讯、钢联数据 |
| 库存数据 | 行业协会、上市公司公告 |
| 销量数据 | 乘联会（汽车）、中怡康（家电）等 |

## 六、行业分类与跟踪

### 6.1 行业分类体系（一级+二级）

**申万一级行业（31个）→ 二级行业（134个）**

采用「一级定方向，二级定标的」的筛选策略：

| 一级行业 | 重点关注二级行业 | 筛选理由 |
|---------|----------------|--------|
| 食品饮料 | 白酒、调味品、乳制品 | 品牌护城河深、提价能力强 |
| 医药生物 | 化学制药、医疗器械、CXO | 创新驱动、老龄化受益 |
| 家用电器 | 白色家电、小家电 | 格局稳定、龙头集中 |
| 电子 | 半导体、消费电子、PCB | 国产替代、周期底部 |
| 电力设备 | 光伏、锂电池、储能 | 政策支持、成长空间大 |
| 汽车 | 乘用车、汽车零部件 | 电动化智能化趋势 |
| 机械设备 | 工程机械、通用设备 | 制造业升级受益 |
| 银行 | 股份行、城商行 | 低估值、高股息 |

**规避的二级行业**：
| 一级行业 | 规避二级行业 | 规避理由 |
|---------|------------|--------|
| 房地产 | 房地产开发 | 政策压制、需求下滑 |
| 传媒 | 游戏、影视 | 监管不确定、竞争激烈 |
| 社会服务 | 教育、旅游 | 政策影响大 |
| 纺织服饰 | 纺织制造 | 产能过剩、低毛利 |

### 6.2 行业池分层管理
按照申万一级行业分类，建立跟踪池：

**核心优选行业**（长期配置）：
- 食品饮料、医药生物、家用电器
- 电子、计算机、通信
- 电力设备、国防军工

**机会关注行业**（周期配置）：
- 银行、非银金融、房地产
- 有色金属、煤炭、化工
- 汽车、机械设备

**规避行业**（短期规避）：
- 政策不确定性高的行业
- 竞争格局恶化的行业
- 景气度持续下行的行业

### 6.3 行业跟踪机制
**月度跟踪**：
- 行业经营数据（产量、销量、价格等）
- 行业财务指标（收入、利润、ROE等）
- 行业政策动态与事件追踪

**季度评估**：
- 行业景气度评分更新
- 行业配置权重调整建议
- 行业内个股优选调整

**年度复盘**：
- 行业筛选标准有效性验证
- 行业池动态调入调出
- 行业投资策略优化

## 七、行业筛选决策流程

### 7.1 决策步骤
```
Step 1: 政策环境评估
        ↓ （通过）
Step 2: 商业模式评估
        ↓ （通过）
Step 3: 竞争格局评估
        ↓ （通过）
Step 4: 景气度评估
        ↓ （通过）
Step 5: 纳入目标行业池
        ↓
Step 6: 确定配置权重
```

### 7.2 评分矩阵（参考）
对每个行业进行打分（1-5分），总分 > 15分纳入目标池：

| 评估维度 | 权重 | 评分标准 |
|---------|------|---------|
| 政策环境 | 20% | 5分：强力支持；3分：中性；1分：限制 |
| 商业模式 | 25% | 5分：优秀；3分：良好；1分：较差 |
| 竞争格局 | 25% | 5分：寡头垄断；3分：集中；1分：分散 |
| 景气度 | 20% | 5分：高景气；3分：平稳；1分：下行 |
| 护城河 | 10% | 5分：深厚；3分：一般；1分：薄弱 |

## 八、动态调整与风险管理

### 8.1 调入触发条件

**行业调入信号**（满足2条以上可考虑调入）：
| 信号类型 | 具体条件 | 确认方式 |
|---------|---------|--------|
| 政策转向 | 重大利好政策出台 | 政策文件、高层讲话 |
| 景气反转 | PMI连续3月回升且突破50 | 统计局月度数据 |
| 估值底部 | PE/PB处于历史10%分位以下 | Wind/理杏仁 |
| 供给出清 | 产能利用率企稳回升 | 行业协会数据 |
| 龙头改善 | 行业龙头业绩超预期 | 财报、业绩快报 |

### 8.2 调出触发条件（红线机制）

**立即调出**（触发任一条件）：
| 红线指标 | 触发条件 | 监控频率 |
|---------|---------|--------|
| 政策风险 | 重大利空政策（如行业整顿） | 实时 |
| 竞争恶化 | 行业毛利率连续4季度下滑>5pct | 季度 |
| 需求崩塌 | 行业收入连续2季度同比下滑>20% | 季度 |
| 估值泡沫 | PE处于历史95%分位以上 | 月度 |

**预警观察**（触发后加强监控）：
| 预警指标 | 触发条件 | 后续动作 |
|---------|---------|--------|
| 盈利下滑 | 行业ROE连续2季度低于8% | 降低权重至50% |
| 景气见顶 | PMI连续3月回落且跌破50 | 准备退出计划 |
| 库存积压 | 库存周转天数上升>30% | 暂停新增配置 |
| 资金流出 | 北向/主力连续4周净流出 | 评估是否减仓 |

### 8.3 行业筛选的局限性
- 行业整体向好不代表所有个股都能获益
- 行业数据存在滞后性，需结合前瞻指标
- 短期波动可能偏离长期逻辑

### 8.4 年度复盘机制
- 回测筛选标准有效性，调整不合理的阈值
- 审视调入/调出决策的准确率
- 分析错判案例，优化判断框架
- 更新行业池（调入新兴行业，调出衰退行业）

---

## 九、总结

行业筛选是价值投资的第一道关口，通过系统化的筛选框架：
1. **剔除**：政策风险高、商业模式差、产能过剩的行业
2. **保留**：格局稳定、需求确定、具备护城河的行业
3. **优选**：景气度向上、受益产业升级、估值合理的行业

通过行业筛选后，进入[个股筛选](./2：个股筛选.md)环节，在优质赛道中挑选优质公司。
